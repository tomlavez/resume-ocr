Procuramos um Analista de Dados para extrair insights valiosos de nossas fontes de dados. O profissional deve ter forte conhecimento em SQL e Python para análise de dados (bibliotecas Pandas, NumPy). Experiência em criar visualizações e dashboards em ferramentas como Tableau ou Power BI é essencial. Conhecimento em modelos de machine learning com Scikit-learn é um diferencial importante.
